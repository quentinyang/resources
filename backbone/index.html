Learning Backbone

Learning 1: This example illustrates the declaration and instantiation of a minimalist View.

Learning 2: This example illustrates the binding of DOM events to View methods.

Learning 3: This example illustrates how to use a Collection of Models to store data, and how to tie changes in those to a View.

Learning 4: This example illustrates how to delegate the rendering of a Model to a dedicated View.

Learning 5: This example introduces two new Model actions (swap and delete), illustrating how such actions can be handled within a Model's View.
